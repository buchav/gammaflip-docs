---
title: 'List Coins'
api: 'GET /coins'
description: 'Returns available base coins across all exchanges'
---

## Response

<ResponseField name="data.coins" type="array">
  Sorted list of base coin symbols (uppercase) that have options listed on at
  least one supported exchange.

  Typical values: `BTC`, `ETH`, `SOL`, `XRP`, and others that exchanges may list.
</ResponseField>

<RequestExample>

```bash cURL
curl https://api.gammaflip.io/api/v1/coins \
  -H "X-API-Key: gex_sk_your_key_here"
```

```python Python
import requests

resp = requests.get(
    "https://api.gammaflip.io/api/v1/coins",
    headers={"X-API-Key": "gex_sk_your_key_here"},
)
coins = resp.json()["data"]["coins"]
print(f"Available coins: {', '.join(coins)}")
```

</RequestExample>
